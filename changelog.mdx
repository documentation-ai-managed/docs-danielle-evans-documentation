<Update label="2024-10-15" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Introduced full MDX v3 support with advanced JSX components like `<Steps>` and `<Tabs>`.
- Added brand color customization (`#3B82F6` default for Danielle Evans theme).

## Improvements

- Enhanced parsing rules prevent common errors like unescaped `{` or `<` characters.
- Sidebar navigation now supports dynamic frontmatter titles.

## Breaking Changes

- Updated Mermaid syntax to code blocks only—no more `<Mermaid>` component.

## Bug Fixes

- Fixed YAML frontmatter parsing with strict no-blank-line enforcement.

```javascript
// Example: New component usage in v1.0.0
<Steps>
  <Step title="Install">npm install danielle-docs</Step>
</Steps>
```

</Update>

<Update label="2024-09-20" description="v0.9.0" tags={["feature", "improvement", "bugfix"]}>

## New Features

- Added `<Update>` component for changelog entries with `tags` support.
- Integrated Lucide icons library for all visual elements.

## Improvements

- CodeGroup now auto-detects languages without manual `tabs` specification.
- Performance boost: Lazy loading for expandable sections.

## Bug Fixes

- Resolved nested code block issues by enforcing 4-backtick syntax.
- Corrected attribute quoting—always double quotes for strings.

<CodeGroup tabs="JavaScript,Python">
```javascript
// Upgraded auth example
const auth = await fetch('/api/docs', {
  headers: { Authorization: `Bearer ${token}` }
});
```

```python
# Upgraded auth example
import requests
response = requests.get('/api/docs', headers={'Authorization': f'Bearer {token}'})
```
</CodeGroup>

</Update>

<Update label="2024-08-10" description="v0.8.0" tags={["feature", "bugfix"]}>

## New Features

- Launched `<ParamField>` and `<ResponseField>` for API documentation.
- Support for `<Columns>` layouts with up to 5 columns.

## Bug Fixes

- Fixed heading hierarchy enforcement—no more H1 in body content.
- Escaped special characters in prose now render correctly.

</Update>
